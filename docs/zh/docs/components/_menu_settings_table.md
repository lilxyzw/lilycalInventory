| 名称                   | 说明                                    |
|----------------------|---------------------------------------|
| 图标                   | 指定所创建菜单的图标。                           |
| 菜单&参数名               | 指定所创建菜单和参数的名称。                        |
| 菜单覆盖（Modular Avatar) | 使用Modular Avatar配置的Menu Item来管理上述设置项。 |
| 保存状态                 | 指定在切换Avatar或移动世界时是否保持所创建菜单的开关状态。      |
| 仅限本地                 | 指定所创建菜单的开关状态是否只在本地生效。                 |
